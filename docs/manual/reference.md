| Key | Action |
|---|---|

## Tips

- **Layer sounds** — Multiple tracks can use the same instrument. Stack a bass and a pad on the same synth, or double a melody an octave apart.
- **Mute to focus** — Mute tracks you're not editing so you can hear a part in isolation. Muted tracks are also excluded from WAV exports.
- **Audition a single track** — Switch to **Track** mode in the transport, select a track, and loop it to hear just that part cycling.
- **Edit the presets** — The built-in sounds are starting points. Open any instrument and nudge the ADSR sliders until it sits right in the mix.
- **Set the loop before exporting** — The WAV export uses the loop region. Always check the loop start/end before clicking WAV.
- **Single-click to place notes quickly** — After drawing one note, every subsequent click places a note at the same length. You only need to drag when you want a different size.

## Limitations

| Area | Limitation |
|---|---|
| Arrangement length | Fixed at **32 bars**. Cannot be extended. |
| Clip length | All new clips are **4 bars**. Length cannot be changed after placement. |
| Undo | **No undo system.** Deletions of notes, clips, and tracks are permanent. |
| Note velocity | Fixed at a single level. Per-note velocity editing is not available. |
| Effects | No reverb, delay, EQ, or other effects in the current version. |
| Zoom | No zoom in the arrangement or piano roll. |
| Clip resize | Clips cannot be resized or moved after placement — remove and re-place. |
